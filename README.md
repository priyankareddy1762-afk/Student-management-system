# 🎓 Student Management System

A simple Java console-based application to manage student records. This project allows users to add and display student information using object-oriented programming concepts.

## 🚀 Features

- ➕ Add Student
- 📋 Display All Students
- 🖥️ Menu-driven interface
- 💻 Console-based application

## 🛠️ Technologies Used

- Java
- Visual Studio Code
- Git & GitHub

## 📂 Project Structure

```
Student-Management-System
│── Main.java
│── Student.java
│── StudentManager.java
```

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Student-Management-System.git
```

2. Open the project in Visual Studio Code.

3. Compile the Java files.

```bash
javac *.java
```

4. Run the program.

```bash
java Main
```
